Mobile

Projects or products developed and supported by Miing.org are exactly around the mobile industry.

Android

The Android project is for developing android based smartphone solutions powered by chipsets from Qcom.

For more information, go over to the Android homepage.

LMKP

The LMKP project aims to enable chipsets from Qcom on Linux Kernel on which to run Android.

For more information, take a look at the LMKP homepage.

Bootloader

The Bootloader project is to boot up Linux Kernel by Little Kernel.

For more information, have a look at the Bootloader homepage.

Aurora

The Aurora project is to develop and implement mobile protocol stack for modem powered by chipsets from Qcom.

For more informations, go look at the Aurora homepage.
