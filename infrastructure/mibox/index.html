What's MiBox?

MiBox is a software stack for software teams deploying an infrastructure for development, quality assurance and delivery of software between developers and users in the most efficient way.

Features

  • Single Sign On enabling identity management based on OpenID
  • Project Management managing all projects launched and keeping track of all features, issues, and bugs
  • Code Review and Management examining and storing source code
  • Continuous Integration automatically building, testing and reporting source code
  • Collaborative Documentation Sharing allowing to add, modify and share stuff
