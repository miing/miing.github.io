Infrastructure

Infrastructure groups together a set of open source software to build up a software development environment in the way of software engineering.

MiBox

MiBox is used to help software teams deploy a software development infrastructure and draw up a DevOps workflow to work together in the most efficient way.

For more information on it, go over to the MiBox homepage.

MiGo

MiGo is exactly a Single Sign-On system that manage and identify identities based on OpenID throughout all services on infrastructure. In other words, one account throughout all services.

For more information about it, take a look at the MiGo homepage.
